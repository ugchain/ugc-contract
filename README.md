# Directory Structure
```
├─ugc-contract # truffle编译智能合约代码
│   ├─abi.json # 生成的abi文件
│   ├─contracts # 各个合约代码的存放处
│   │─migrations # 工具包
│   ├─test # 测试脚本
│   ├─truffle-config.js # 全局配置文件
│   ├─truffle.js # 部署文件
│   ├─build # 生成的合约文件
```
